pr 3 - Create a Map: 

A. Working with Attributes.
Steps--

1. Open a Thematic Map. Go to Add Layer - Vector Layer - Practical 3 -> India_State_adm1  
Import all Maps of Maharashtra.

2. Go to Project → New PrintLayout 
• Insert a suitable title and press “OK”. 
• A new Print Layout window will open. 

3. Select Add Item → Add Map 

4. After adding map go to ItemProperties → Map1→ Layers Check on Lock Layers and Lock Styles for Layers. 

5. Go to Add Item → Add Map → Place a Map box at appropriate location.

6. Set a frame for Inset by enabling the check box for Frame in Item Properties.

7. To highlight the area shown in Inset,

   Select the Picture representing main Map from Items pane. 
   In Item Properties → Overviews → using icon add an overview.

8. Select the checkbox Draw Overview. Name the Picture object representing inset (Map1 in our case).

9. Add Item → Add Label
   
    Change the Label text To “Mumbai Map”, Set appropriate font size and color using Item Properties→ Main Properties. Also add a text saying “Copyright Reserve”

10. Add Item → Add Legend→ Place the legend indicator at appropriate location. 

11. Add Item → Add Scale Bar

12. Add Item → Add North Arrow

13. A Map can be saved in Image or PDF using Layout → Export as Image / Export as PDF

Finished.
